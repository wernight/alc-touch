Présentation du programme

Ce programme est destiné à remplacer les fonctions DOS existantes dont l'utilisation n'est pas aisée. En plus de cela les fonctions sont améliorées et adaptées à Windows 95/98.

Prise en main du programme

Le programme se présente comme un Wizard où l'on progresse en 3 étapes.

  1. Sélection des fichiers à modifiers

    2. Dans cette partie vous définissez les fichiers à modifiers.
    Commencez par indiquer le répertoire où se trouve les fichiers.
    L'options "Inclure les sous-répertoires" permet de modifers les fichiers du répertoire sélectionné et de modifiers les fichiers se trouvant dans les répertoires de celui sélectionné.
    Les attributs fonctionnent de la façon suivante: Prenons l'exemple de l'attribut "Lecture seule", si il est grisé le programme prend les fichiers ayant cet attribut ou nom. Si il est coché, seul les fichiers possédant cet attribut sont pris. S'il ne l'est pas, seul les fichiers n'ayant pas cet attribut sont pris. Maintenant si vous voulez les fichier ayant l'attibut Archive mais pas l'attribut Système, les autres vous étants indifférents: Cochez Archive, décochez Système et laissez les autres en gris.
    La zone "Spécification" vous permet de filtrer les type de fichiers, leur extention et leur nom. Pour plus d'informations cliquez ici..
    Il se peut que vous souhaitiez encore réduire la sétion. Cochez alors "Fichiers sélectionnés", puis cliquez sur "Rafraichir" ou "Afficher" (selon celui qui apparaît. Vous aurez une liste des TOUS les fichiers correspondant à vos critères précédents. A côté de chaque fichier à modifier cocher la case.
    Vous pouvez aussi utiliser "Rafraichir" ou "Afficher" (selon celui qui apparaît) pour voir la liste des fichiers correspondant à vos critères.

  2. Opérations à effectuer sur les fichiers

    3. Vous dévinissez ici les opérations à effectués sur les fichiers sélectionnés dans la partie d'avant. Vous pouvez effectuer simultanément plusieurs modification en cochant la case à gauche du nom de la modification à faire. Il existe 4 type de modification:
    1. Date & Heure : Chaque fichier Windows possède 3 dates. La date de création du fichier, la date de la dernière modification et du dernier accès (lecture) du fichier. Cochez la case à gauche des date à changer et entrez la nouvelle date ou demander à placer la date courante.
    2. Case : Depuis Windows 32bits les fichiers peuvent être en majuscule ou minuscule: C'est la case. Cependant L'explorateur Windows n'affiche pas forcément les fichers qui sont qu'en majuscule en majuscule.
    3. Renommer : Commencez par donner les spécifications du nom de départ dans la zone de texte du haut dans la zone "Renommer". Cette première étape permet de filtrer encore une fois les fichiers, mais surtout vous devez placer un chiffre (0123456789) après chaque "*" ou "?". Si vous mettez "*1" dans la première zone, la où vous placerez "*1" dans la seconde zone (si vous le mettez), il sera remplacé par ce qui se trouvait à la place de "*1" dans la première zone.
      Exmple: Vous voulez changer toute une suite de documents ayant l'extention "doc", commençant par "Compte", finissant par "2000" ou "2001" ou plus. Mais vous ne voulez pas changer les fichier "Compte2000.doc". Dans ces fichiers vous voulez changer le "2000" ou "2001"... en "00" ou "01" ...
      Extrez dans la première zone "Compte*1?1?2?3?4?5?6.doc" et dans la seconde "Compte*1?1?3?4.doc".
      Les changement seront alors:
      Compte2000.doc -> Compte2000.doc
      Compte25-01-2000.doc -> Compte25-01-00.doc
      Compte25-01-2001.doc -> Compte25-01-01.doc
      etc. -> etc.
    4. Attributs : Changez ici les attributs des fichiers. Elle fonctionne presque comme celle de la 1ere partie à la différence qu'ici il ne s'agit plus de filtrer mais de changer mes attributs.
      Les attributs en gris ne seront pas changés. Si vous cochez un attribut, il sera mis, si vous décochez un attribut il sera enlevé.

  3. Exécuter les modifications

    4. Vous pouvez à tout moment aller voir cette partie pour voir ce que vous n'avez oublié de définir. Si des modifications peuvent être effectuées, vous n'avez qu'à cliquer sur le gros bouton central pour lancer les modifications. C'est seulement à ce moment que les fichiers sont changés.
    Si vous voulez aller plus vite, cliquez sur "Exécuter" en bas à gauche.

Comment ça marche les zones ou l'on spécifit le nom d'un fichier ?

Les spécifications dans "Renommer" permettent l'utilisation de plusieurs "*" et/ou "?" dans chaque spécification. En plus de cela vous devez y placer un chiffre après chaque "*" ou "?".
Dans toutes les zones de spécification les "*" représentent n'importe quel caractère ou suite de caractères, voir même aucun caractère. Les "?" représentent un caractère n'importe lequel.
(Les exemples ci-dessous marchent partout sauf dans pour "Renommer" où il faut un chiffre avec les * et ?)
Vous devez mettre un "." pour séparer le nom de l'extention (sauf Renommer). Finissez par ".*" si l'extention vous est égale ou inconnue. Si vous voulez connaître l'extention d'un fichier, cliquez droit sur ce fichier dans l'Explorateur Windows par exemple puis "Propriétés", vous la verrez en haut dans la fenêtre qui s'affiche normalement. "*.*" représente tous les fichiers, n'importe leur nom et leur extention. Si vous voulez que les images Bitmap commençant par "Img", entrez dans cette zone: "Img*.bmp"
